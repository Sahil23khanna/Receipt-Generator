Receipt Generator

Receipt Generator is a simple web application that helps users calculate
the total bill amount based on item prices, quantities, and applicable charges.

The project allows users to enter multiple items and automatically calculates
the final amount, making it useful for small shops, billing practice,
and learning basic calculation logic in web development.


Tech Stack :
- JavaScript
- HTML
- CSS


Key Features :
- Add multiple items with price and quantity
- Automatic total calculation
- Simple and user-friendly interface
- Real-time calculation updates

Purpose of the Project :
This project was built to practice frontend development concepts
and JavaScript logic such as event handling, calculations,
and dynamic UI updates.
